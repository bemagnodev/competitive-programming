Repositório pessoal para arquivar soluções de problemas de programação competitiva. O objetivo é manter um histórico de evolução dos meus estudos para competições em algoritmos e estruturas de dados.

## Estrutura do Repositório

As soluções estão organizadas automaticamente por pastas, dependendo da plataforma de origem:

- **Codeforces**: Soluções de contests (Div 2, Div 3, Educational) e prática geral.
- **CSES**: Soluções do CSES Problem Set.
- **Uncategorized**: Problemas diversos (OBI, AtCoder, USACO , etc).

## Workflow e Automação

Utilizo um script em Python desenvolvido localmente para organizar a estrutura de pastas. O script identifica a origem do problema baseada no padrão do nome do arquivo (ex: `1702E.cpp` para Codeforces) e move para o diretório correto, preservando a data original de modificação do arquivo no histórico do Git.
